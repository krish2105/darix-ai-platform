**DARIX AI** is a premium, highly interactive SaaS platform designed to evaluate and measure the Artificial Intelligence readiness of businesses operating in Dubai, UAE. Built as a Master's degree showcase in AI Business, it blends stunning 3D-like glassmorphism aesthetics, fluid motion graphics, and advanced readiness logic.

## 🌟 Key Features
- **Interactive AI Readiness Assessment**: An 8-dimension business intelligence form.
- **Dynamic Scoring Algorithm**: Real-time scoring based on data maturity, governance, and automation potential.
- **Flawless Light/Dark Mode**: Completely semantic CSS architecture using Tailwind CSS and `next-themes`.
- **Premium Animations**: Floating data windows and a neural network SVG background powered by Framer Motion.
- **Executive Report Generation**: Simulated PDF data structures for board-ready business insights.

## 🚀 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/darix-ai-platform.git
   cd darix-ai-platform
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. **Open in browser:**
   Navigate to `http://localhost:3000`

## ☁️ How to Deploy (Going Live!)

The absolute best and easiest way to deploy this Next.js application is through **Vercel**, the creators of Next.js.

### Step 1: Push to GitHub
1. Create an empty repository on GitHub.
2. Open your terminal in the project folder and run:
   ```bash
   git add .
   git commit -m "Initial commit: DARIX AI Platform"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```

### Step 2: Connect to Vercel (Free & Instant)
1. Go to [Vercel.com](https://vercel.com/) and create a free account using your GitHub login.
2. Click the **"Add New..."** button and select **"Project"**.
3. You will see your newly created GitHub repository in the list. Click **Import**.
4. You do not need to change any settings (Vercel automatically detects Next.js).
5. Click **Deploy**.

*Vercel will build the project and provide you with a live HTTPS URL (e.g., `https://darix-ai-platform.vercel.app`) in less than 2 minutes.*
